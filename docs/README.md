### 如何贡献

* 写作

``` bash
# 克隆 repo
git clone +url1

# 开发预览
npm run dev

```

* 提交

所有变更可直接提交到 develop 分支, 项目负责人审核过后,会合并到 master

::: warning 注意
请确保 Node.js 版本 >= 8.6。
:::
